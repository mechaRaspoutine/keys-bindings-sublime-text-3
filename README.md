# keys-bindings-sublime-text-3
usefull keybindings for sublime text editor

Usage:
  
  You have to change these files on "sublime_text\Data\Packages\User" folder.


Features:

 ctrl + (',' or ';'):
  - go to end of current line
  - add a (',' or ';')

 ctrl + alt + (',' or ';'):
  - go to end of current line
  - add a (',' or ';')
  - new line
